# Delhi Social Network Recommender

## Overview

Delhi Social Network Recommender is a Python-based project that simulates a social networking platform and generates recommendations for users.

The project analyzes friendship connections and page interests to provide intelligent recommendations such as:

* People You May Know
* Pages You Might Like
* Mutual Friend Analysis
* Interest-Based Recommendations
* Community Discovery

## Problem Statement

Social media platforms use recommendation systems to help users discover relevant people and communities.

This project demonstrates how graph relationships and user interests can be used to build recommendation engines.

## Dataset

The dataset contains:

### Users

* User ID
* User Name
* Friend Connections
* Liked Pages

### Pages

* Page ID
* Page Name
* Community Information

The project works with JSON datasets representing social network relationships.

## Features

### People You May Know

Recommends users based on:

* Mutual friends
* Shared interests
* Network proximity

### Pages You Might Like

Suggests pages using:

* Friend preferences
* Similar interests
* Community engagement

### Data Cleaning

The project also includes preprocessing and cleaning steps for handling:

* Duplicate records
* Missing values
* Invalid relationships
* Dataset normalization
